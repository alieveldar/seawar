React + redux app for counting Russian Tax (НДФЛ) 
=====================

### Usage

Install node,js (depends on your OS)
https://nodejs.org/

```
npm install
npm start
open http://localhost:3000
```
### Technologies
- ES6 (babel)
- webpack
- React.js
- Redux
- react hot reload
